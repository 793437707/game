# 游戏类型
第三人称视角的联机对战游戏
# 开发工具
- Visual Studio 2022
- Unreal Engine 5.2.1
- Git/P4
# 游戏内容
## 核心玩法
游戏含有多个可选职业，玩家可根据自己的职业[^1]，进行额外属性的搭配[^2]，传入同一张地图，进行对战，另一方倒下后己方获胜，时间结束时根据剩余血量或得分方式，判决胜利。
## 其他功能[^3]
- 玩家的登录注册，服务器保留玩家数据。
- 好友和聊天功能。
- 排行/房间。
